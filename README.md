# Arc

## Description

A bytecode-compiled programming language with a virtual machine written in [Rust](https://www.rust-lang.org/).

## Instructions

You will need [Cargo](https://doc.rust-lang.org/cargo/) installed to compile and run Arc.

### Run the Repl

```bash
cargo run
```

### Run a Specific File

```bash
cargo run <file_name.arc>
```

### Run the Tests

```bash
cargo test
```
